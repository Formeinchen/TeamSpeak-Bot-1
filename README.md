### Installation

Install Java

```sh
$ sudo apt update
$ sudo apt upgrade
$ sudo apt install default-jre
$ java -version
```

Download the Bot

```sh
$ wget 
```

Run the Emulator

```sh
$ chmod 777 start.sh
$ ./start.sh run
```
